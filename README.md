# SQL Superstore Sales & Profit Analysis

## Objective
Analyze sales and profit data to identify which products, regions, categories, and customer segments the company should target or avoid.

## Tools Used
- PostgreSQL
- SQL
- Vscode

## Key Business Questions
1. Which region and customer segment generate the most profit?
2. Which sub-categories perform poorly by profit margin?
3. Which areas show strong performance and should be expanded?
4. Which products (e.g., Tables) can be replicated due to high margins?
